# Dream Arts Coding Test

## 概要
鉄道網をグラフとして扱い、同じ駅を2回通らない経路の中で、合計距離が最大となる経路を求めるプログラムです。

## 入力形式
各行に以下の形式で与えられます。

始点ID, 終点ID, 距離

例：
1, 2, 8.54
2, 3, 3.11
3, 1, 2.19
3, 4, 4
4, 1, 1.4

## 出力形式
最長経路となる駅IDを通過順に1行ずつ出力します。

## 実行方法
以下のコマンドで実行できます。

python3 main.py < input.txt

## 解法
入力を重み付き無向グラフとして構築し、すべての駅を始点として深さ優先探索を行います。

探索中は訪問済みの駅を管理し、同じ駅を2回通らないようにします。各経路の距離を計算し、最大となる経路を更新します。

## 計算量
最長単純路問題は一般グラフではNP困難であり、本実装ではDFSによる全探索を行うため、計算量は指数時間となります。
